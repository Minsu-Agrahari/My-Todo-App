# 📝 Todo App

A simple and clean **Todo Application** built using **HTML, CSS, and JavaScript**.  
This project helps you manage your daily tasks with ease and a smooth user experience.

---

## 🚀 Features

✨ Add new tasks  
🗑️ Delete tasks with animation  
📊 Live task counter  
⌨️ Add task using **Enter key**  
🎨 Clean and modern UI  
⚡ Smooth animations for better UX  

---

## 🛠️ Tech Stack

- 🌐 HTML  
- 🎨 CSS  
- ⚙️ JavaScript  

---

## 📸 Preview

A minimal dark-themed Todo App where you can:
- Add tasks instantly  
- Remove tasks with animation  
- Track total number of tasks

## ⚙️ How It Works

1. ✍️ Enter a task in the input field  
2. ➕ Click **Add** button or press **Enter**  
3. 📋 Task gets added to the list  
4. 🗑️ Click **Delete** to remove a task  
5. 🔢 Task counter updates automatically

## 🆕 Recent Updates

- Tasks are displayed in a horizontal layout with Delete button beside each task
- Double-click a task to edit it in-place; press Enter or click away to save
- All layout and editing styles are handled directly in JavaScript (inline styles)
- Task editing and deletion animations remain unchanged
